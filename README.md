প্রতিষ্ঠান পরিচিতি,
 প্রতিষ্ঠানের পাঠদানের অনুমতি ও স্বীকৃতি, এমপিও,
  শ্রেণি ও লিঙ্গভিত্তিক শিক্ষার্থীর তথ্য,
   শ্রেণিভিত্তিক অনুমোদিত শাখার তথা,
    পাঠদান সংক্রান্ত তথ্য (শিক্ষক,শিক্ষিকার নামসহ পূর্ণাঙ্গ রুটিন,
      পাঠ্যসূচী, বিবিধ নোটিশ ইত্যাদি),
        প্রতিষ্ঠানের ফোন/মোবাইল নম্বরসহ যোগাযোগের ঠিকানা,
         তথ্য সেবা কেন্দ্রের ঠিকানা ও মোবাইল নম্বর,
          অভিযোগ নিষ্পত্তি কর্মকর্তার ঠিকানা ও মোবাইল নম্বর,
           প্রতিষ্ঠান প্রধানসহ সকল শিক্ষক-কর্মচারীর তথ্য,
            ব্যবস্থাপনা কমিটির তথ্য
